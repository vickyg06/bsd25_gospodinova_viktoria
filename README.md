# Text Encryption & Decryption Java Programm

## Brief Description of the Project
This project is a console-based Java application that encrypts and decrypts text using a custom rotational cipher (user input). The program shifts characters forward or backward based on a user-defined numeric key (rotation). 

**Key Features:**
+ **Alphanumeric Rotation:** Specifically handles uppercase letters (`A-Z`), lowercase letters (`a-z`), and digits (`0-9`). 
+ **Safe Punctuation:** Special characters, spaces, and punctuation marks remain completely untouched.
+ **Interactive Menu:** A continuous loop menu that allows users to perform multiple encryptions or decryptions without restarting the program.
   ```
   ------------------------------------------------------------------------
   1 - Encrypt text
   2 - Decrypt text
   9 - Quit
   ------------------------------------------------------------------------
---

## Installation & Usage Instructions

### Preparation
* You must have the [Java Development Kit (JDK)](https://www.oracle.com/java/technologies/downloads/) installed on your system.

### Installation
1. Download or copy the source code and save it in a file named `Encryption.java`.
2. Open your terminal or command prompt.
3. Navigate to the directory where you saved the file.
4. Compile the program using the following command:
   ```
   javac Encryption.java
**OR** Install IntelliJ, open the `Encryption.java` and click run.
