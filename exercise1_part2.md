### Git Commands
+ git config - set or view Git configuration.
+ git init - initialize a new Git repository in the current folder
+ git status - shows the current state of the files and changes
+ git add - states changes to be included in the next commit
+ git log - display the commit history.
+ git diff - show differences between files or commits.
+ git pull - fetch and merge changes from the github repository.
+ git commit - saves staged changes 
+ git push - uploads local commits to the github repository
+ git branch - create, list, or delete branches.
+ git checkout - switch branches or restore files.
+ git merge - combine changes from one branch into another.

### Observations on git status and .gitignore
After creating the `.gitignore` file and adding `*.html` and `*.pdf` to it, I ran `git status`. The HTML and PDF files completely disappeared from the untracked list.
